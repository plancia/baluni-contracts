
# Baluni V1

Minimal documentation on Baluni V1 is available via docegen folder.

The built contract artifacts can be browsed via artifacts folder.

## Local Development

The following assumes the use of node@>=10.

### Install Dependencies

yarn

### Compile Contracts

yarn compile

### Start Chain

yarn chain

### Run Tests

yarn test
